¿QUÉ ES BALP?

"Basic Lenguage Programming" es un lenguaje de programación sencilla, creado con una sintaxis simple cuyo objetivo es conservar las reglas básicas de programación cambiando su sintaxis, es un pequeño proyecto que permite validar variables, constantes, estructuras condicionales y ciclos repetitivos.

SINTAXIS

VARIABLES Y CONSTANTES

Para declarar y mostrar variables y constantes la sintaxis es la siguiente. Solo se utiliza NUMBER para una variable númerica y STRING para cadena.

a=5
MOSTRAR a
NUMBER b = 4
MOSTRAR b,"es una prueba"

CONDICIONALES

Para crear una estructura condicional en BALP (If, ElseIf y Else) la sintaxis es la siguiente.

a = 6
b = 6
CUANDO a==b HACER
MOSTRAR "Los numeros son iguales"
FNCUANDO

CICLO FOR

Para crear una estructura repetitiva FOR en BALP la sintaxis es la siguiente.

CICLOP j=1 LIMITE 6
MOSTRAR j
FNCICLOP

OPERACIONES BÁSICAS

Para crear una hacer eperaciones básicas en BALP la sintaxis es la siguiente.

a = 17
b = 8
"La suma es:", a + b
"La resta es:", a - b
"La multiplicación es:", a * b
"La división es:", a / b

CADENAS Y ENTERO

Para declarar, concatenar y mostrar una cadena en BALP la sintaxis es la siguiente.

NUMBER edad
edad = 15
MOSTRAR "Juan tiene",edad,"años."

CICLO WHILE

Para crear una estructura repetitiva WHILE en BALP la sintaxis es la siguiente.

i = 0
CICLOMQ i<5
i=i+1
MOSTRAR i
FNCICLOMQ

SOBRE EL PROYECTO

BALP es un proyecto de Compilador, es un lenguaje de programación netamente con Java Script para hacerlo funcional en la web, este compilador analiza el lexer y el paxer de una instrucción y la compila validando si la expresión es correcta o no segun el lenguaje de programación BALP.
